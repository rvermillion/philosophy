# Welcome to Liminalism

This site is (appropriately as you will see) a work in progress.  It's a place for me to collect my thoughts and writings on **Liminalism**, the personal, practical philosophy that I've been working on for the last six years.

One part of the work, very much still in progress, is to organize and present the ideas better.  As part of that, I've started trying to add some structure to each component: laying out the major ideas, adding *interludes* that reflect on them, proposing *practices* that can help integrate the ideas into daily life, and asking *questions* that will help with reflection and introspection on the topic.  Much of this is waiting to be done.
